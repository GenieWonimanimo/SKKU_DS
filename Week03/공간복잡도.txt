공간복잡도에 대하여
float sum(float *list, int n)
{
    float tempsum = 0;
    for (int i = 0; i < n; i++)
        tempsum += *(list + i);
    return tempsum
}

float rsum(float *list, int n)
{
    if (n > 0)
        return rsum(list , n - 1) + list[n - 1];
    else
        return 0;
}

첫번째 함수는
공간 복잡도가 매개변수에서 8 + 4바이트로 12바이트,
함수 내에서 지역변수로 4 + 4 바이트로 8바이트
총 20바이트이고
n의 값에 관계없이 항상 20바이트 이므로
O(1) 이라 할 수 있다.

두번째 함수는
공간 복잡도가 매개변수에서 8 + 4바이트로 12바이트
list[n-1] 을 위한 상수에 대한 공간 C1바이트로
12 + C1 바이트 * 재귀 수 이다.
n의 값에 따라 달라지고, 12 * n
이기 때문에 O(n) 이라 할 수 있다.
